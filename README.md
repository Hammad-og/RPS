# 🎮 Rock Paper Scissors Telegram Bot

A simple, fun telegram bot to play rock paper scissors!

## Features

✅ Play against the bot  
✅ Track your stats  
✅ Global leaderboard  
✅ Works in groups and topics  
✅ Runs 24/7

## Quick Start

1. Get bot token from @BotFather on Telegram
2. Deploy to Railway
3. Add BOT_TOKEN to Railway variables
4. Start playing!

## How to Play

- Search your bot on Telegram
- Click /start
- Click buttons: 🪨 Rock, 📄 Paper, ✂️ Scissors
- View stats with /stats
- Check leaderboard with /leaderboard

## Deployment

Deploy to Railway:
1. Go to railway.app
2. Connect your GitHub repo
3. Add BOT_TOKEN variable
4. Done!
